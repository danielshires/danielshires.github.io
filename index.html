Main Elements

This section outlines the main elements used on a web page.

Tag Name Description
<html> HTML Wraps the head and body of a webpage
<title> Title Adds the title of the webpage to a browser tab (also used for SEO)
<head> Head Used to content metadata about a webpage
<body> Body Wraps the web pages content
<div> Div Used to create blocks of content on a web page
<h1> Heading 1 The main heading element on the page
<p> Paragraph Paragraph tag used for body copy on the page
<a> Anchor Link an element to another page or website
<img> Image Creates an image element on the page

Table

This section outlines the elelements used for the creation of tables.

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table datacontainer.
<tfoot> Table Dooter The set of rows defining the footer in a table.
